Expresiones faciales a modelos 3D

¿Cómo agregar expresiones faciales 2D a un modelo 3D?.
En esta pagina te explico como lo implementé en mi juego ‘Mini Room Painter’ 👇

Índice:

  1. Preparar el set de expresiones.
  1. Crear el material.
  1. Plano para proyectar las figuras.
  1. Calcular el Tilling.
  1. Posicionar el plano en el personaje.
  1. Crear un controlador.

El objetivo es agregar expresiones faciales a este modelo 3D, utilizando el motor de videojuegos Unity.

1. Preparar el set de expresiones

Utilizaremos un set de expresiones 2D como este:

Necesitamos recortar y posicionar cada figura una detrás de la otra, procurando que todas tengan el mismo tamaño y un pequeño espacio entre cada una.

El tamaño de las figuras deben ser iguales, aunque su contenido no ocupe totalmente el espacio. Esto debe ser así para luego cambiar entre figuras utilizando un desplazamiento fijo.

No es necesario particionar las figuras en el Sprite Editor de Unity, ya que utilizaremos la imagen entera.

2. Crear el material

Creamos un material, asignamos la textura a el BaseMap y configuramos la superficie como transparente.

3. Plano para proyectar las figuras

Modelamos rápidamente un plano donde proyectar las figuras, podemos utilizar el típico ‘Plane' de Unity, o bien, hacer un modelo un poco curvo para adecuarse mejor al personaje.

La curva deformará la textura dando una sensación tridimensional a las expresiones. Esto viene muy bien cuando el personaje es visto de lado, ocultando una parte de la cara.

Luego, asignamos el material que creamos. Además, desactivar la propiedad de CastShadow del Mesh para que no proyecte sombras sobre el entorno.

4. Calcular el Tilling

Ahora necesitamos configurar el Tilling para que encaje con el tamaño de cada figura.

Podemos hacerlo a mano, cambiando el valor hasta que la figura se vea correctamente, o calcularla dividiendo el ancho por figura por el ancho de textura.

Si no sabemos cual es el ancho por figura, lo obtenemos dividiendo el ancho de textura por la cantidad de figuras que contiene la misma.

En este caso, tenemos 16 figuras dentro de una imagen de 7907px de ancho. El ancho por figura es de 494px. Entonces calculamos el Tilling y nos da 0.0625.

Ahora se visualiza correctamente la figura en el plano.

5. Posicionar el plano en el personaje

Posicionamos el plano que acabamos de crear sobre la superficie de el personaje, sobresaliendo de este.

6. Crear un controlador

Por ultimo, creamos un controlador que se encarga de cambiar el offset del material, en base a qué índice de figura queremos mostrar (faceIndex).

public void SetMaterialOffsetByFace(int faceIndex, int textIndex = 0)
{
    // out of bounds or no renderer
    if (faceIndex < 0 || rend == null)
        return;

    // change texture index
    if (textIndex >= 0 && textures != null && textIndex < textures.Length)
        rend.sharedMaterial.mainTexture = textures[textIndex];

    // change material offset
    float offset = materialOffsetSize * faceIndex;
    rend.sharedMaterial.mainTextureOffset = new Vector2(offset, 0);
}

Además, podemos crear una lista de texturas y ampliar la gama de expresiones.
